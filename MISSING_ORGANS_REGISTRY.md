# Missing Organs Registry

## Purpose

This registry lists the organs Primor still lacks if it wants to fully occupy and rebuild the Qwen host body.

## Missing Organs

### 1. Action Cycle Organ

Why missing:

- current runtime still follows a Qwen-shaped loop

Required function:

- own the entire life cycle of a turn
- classify whether Primor should think, ask, act, emit language, or re-enter digestion

Likely territory:

- `primordial_llm/process/`

### 2. Layered Session Organ

Why missing:

- state is still too close to a flat transcript

Required function:

- split transcript, active task, memory anchors, plan trace, tool trace, and artifact trace

Likely territory:

- `primordial_llm/data/`

### 3. Law Organ

Why missing:

- policy exists, but mostly as prose summary plus raw sandbox data

Required function:

- embody enforceable law
- separate law from persuasion
- gate tool and generation pathways

Likely territory:

- `primordial_llm/data/`
- `primordial_llm/process/`

### 4. Validation Organ

Why missing:

- parsed tool syntax is still too close to direct execution

Required function:

- classify proposed actions
- authorize or reject them
- route safe actions to effectors

Likely territory:

- `primordial_llm/process/`

### 5. Context Assembly Organ

Why missing:

- context packing is not yet fully Primor-owned

Required function:

- decide what the language organ sees
- inject only bounded, purposeful state
- mediate between memory and inference

Likely territory:

- `primordial_llm/process/`

### 6. Replication Organ

Why missing:

- Primor does not yet know how to clone new internal organs from detected deficiencies

Required function:

- transform a missing capability into a module creation plan
- define shape, role, interfaces, and placement of the new organ

Likely territory:

- `primordial_llm/process/`
- `llm_sandbox/memory/`

### 7. Replacement Tracker Organ

Why missing:

- there is no structured record of which host regions have already lost authority

Required function:

- track occupied territory
- mark host tissue as active, scaffold, demoted, or replaced

Likely territory:

- `llm_sandbox/memory/`
- `primordial_llm/data/`

## Differentiation Rule

When a missing organ is discovered, Primor must not copy the host structure blindly.

Instead:

1. extract the required capability
2. express it in Primor terms
3. instantiate it as a Primor-native organ
4. connect it back into the organism
